# Abeer Ghafoor · Senior Auditor Portfolio Website

Portfolio preview: https://abeer-ghafoor.netlify.app/

A clean, professional portfolio website for **Abeer Ghafoor**, Senior Auditor at Accountant General Pakistan Revenues (AGPR). This responsive website showcases her professional journey, key expertise, and contact information in an elegant, corporate design.

🔗 **Live Demo:** https://abeer-ghafoor.netlify.app/

##  Features

- **Fully Responsive** - Works perfectly on laptops, tablets, and mobile devices
- **Professional Design** - Clean corporate aesthetic with gold accent colors
- **Experience Showcase** - Detailed work history with icons and skill tags
- **Clickable Contact Info** - Email and phone links with modal popup
- **Smooth Navigation** - Sticky header with section links
- **Fast Loading** - Optimized CSS and vanilla JavaScript


## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Flexbox, Grid, custom properties, responsive queries
- **JavaScript** - Modal interactions, smooth navigation
- **Font Awesome 6** - Professional icons
- **Google Fonts** - Inter & Playfair Display typography
